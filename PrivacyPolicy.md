# Privacy Policy for Auto-Ban

Last updated: 2025-12-14

This Privacy Policy explains how **Auto-Ban** collects, uses, and protects user data.

---

## 1. Data We Collect
Auto-Ban may collect and store the following information:
- Discord User ID (for moderation actions)
- Discord Server (Guild) ID
- Channel ID (for logging purposes)
- Basic bot configuration data

❌ Auto-Ban does **not** collect:
- Message content
- Emails or passwords
- Personal identifiable information outside of Discord IDs

---

## 2. Purpose of Data Collection
Collected data is used only to:
- Detect and ban automated or suspicious bot accounts
- Maintain moderation logs
- Store server-specific settings

---

## 3. Data Storage and Security
Data is stored securely and only for as long as necessary to provide the bot’s functionality.

---

## 4. Data Sharing
We do **not** sell, share, or transfer data to third parties.

---

## 5. Data Removal
Server owners or administrators may request data removal by:
- Removing the bot from their server
- Contacting the bot owner via GitHub or Discord

All related data will be deleted within a reasonable timeframe.

---

## 6. Third-Party Services
Auto-Ban uses the Discord API and complies with Discord’s Privacy Policy:
https://discord.com/privacy

---

## 7. Children's Privacy
Auto-Ban does not knowingly collect data from users under the age of 13, in accordance with Discord’s Terms of Service.

---

## 8. Changes to This Policy
This Privacy Policy may be updated at any time. Continued use of the bot means acceptance of the updated policy.
